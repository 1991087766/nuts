

# 渔夫量化

渔夫量化是一个开源免费的数字货币量化交易系统。使用渔夫量化，你可以轻松接入主流数字货币交易所，从海量开源免费的交易策略中选择策略进行量化交易。你还可以使用手机、平板电脑、PC 等设备监控渔夫量化交易策略的运行状态并根据市场行情随时调整策略参数。
